Getting Started

  1. Log on to the computer and launch a web browser

  2. Navigate to http://loopspace.github.io/LuaCanvas

  3. Click on the checkbox in the lower right corner, labelled Graphics mode

  4. The big area in the centre of the screen is where you can type code (we call this the editor region). Clicking on the box labelled Execute will run your code.

Printing Messages

To display a message, we use the print command. Click in the editor region and type: 


print("hello world")

Now click Execute to run your program.

Click Edit to get back to the program. Try changing the message.

Storing Information

In computer programs, we use variables to store information. A variable is a label which tells the computer what information to use. First we have to store some information. Make sure you are in the editor view and type: 


name = "Harry Potter"

If you run this program, not a lot will happen. That's because we've only stored the information, we haven't done anything with it yet. We can combine this with our earlier message. After the line above, type: 


print("Welcome to Hogwarts,", name)

(Notice the comma after the " and before name.)

Try changing the name and see what happens.

Strings and Numbers

When a computer stores information, it also remembers what type of information it is. A string is a bit like a word; it is made out of letters and other characters, like Harry Potter. Another type of information is a number. By knowing the type of information, the computer is able to do things with that information.

For example, we can take a string and make all its letters lowercase. Following on from the above, type: 


print(name:lower())

What do you think you would get if you changed lower to upper?

If we store a number, we can do numbery things with that information. Type: 


x=3
print(x)
print(x+4)
print(3*x)

(The character between the 3 and x is an asterisk, *. What operation does it represent?)

Getting Information

The last piece of programming that we need today is how to get information. We want to ask the person running the program for information and pass that on to the computer. This means that we want to store it in a variable. Type: 


x = prompt("Enter something:")
print(x)

Run this program. In the text box, type something and press return. Note that the program waits for you to enter something before carrying on.

Planting Vegetables

We can put this together with our algebraic expressions to find out what our gardener would plant. Type the following, but in place of the ?s put the right expression (remember to use * for multiplication): 


m = prompt("How many cabbages did you plant?")
print("You planted",?,"leeks")
print("You planted",?,"carrots")
print("You planted",?,"cauliflowers")
print("Next year, you'll plant",?,"cabbages")

Letters are Unknown Numbers

When writing the program, you didn't know what number m would be. But you could still tell the computer what to do with it, and once the computer found out what m was it could carry out your instructions.

So in the program, m stands for a number that we don't yet know. But as it will become a known number, we can pretend that it is a number and do numbery things to it such as adding, subtracting, and multiplying.
