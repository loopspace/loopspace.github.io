Overview

How the Console Works top ↑

LuaCanvas can run in one of two modes: graphics and non-graphics. In the non-graphics mode, it simulates a console with text input and output. Using this, you can run a more traditional script-style program.

Lua has its own way of getting input and displaying output. The output, using the print function, works as expected, but the way that the input function is implemented in the underlying engine is to use a pop-up. These rapidly get irritating, so LuaCanvas implements a different way with a new function, the prompt function.

Thus a simple program might be:

 print("Welcome to LuaCanvas.")
 name = prompt("What is your name?")
 print("Hello " .. name)

Technical details: The script needs to pause to wait for input without freezing the window. To achieve this, the script is run inside a coroutine which yields when prompt is called and resumes when prompt returns.

IO

print( message ) top ↑

Syntax

print( message )

Displays a message in the output pane. In graphical mode, the output pane is a small panel on the side. In non-graphical mode, the output pane is the whole window. A newline is automatically appended to the string.

message

string to be displayed in the output name

Examples

print("Hello world") x = 3 * 5 print("3 times 5 is " .. x) x,y = 3,5 print(x,y)

prompt( message ) top ↑

Syntax

prompt( message )

This inserts a text box into the output pane and waits for the user to type some input. The user's input is then the return value of the function. If given an argument, this is displayed before the text box. Once the user enters some text and presses return, the text box is disabled to indicate that its purpose is over.

message

(optional) string to be displayed in the output name

Examples

name = prompt("What is your name?") print("Hello " .. name)
